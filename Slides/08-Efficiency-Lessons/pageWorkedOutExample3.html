Proof of f1 ∈ O(f2)

f1(x) = 3 x + 30
f2(x) = x (x + 1) / 2 = 1/2 * (x2 + x)

To prove f1 is an element of O(f2), we have to find real constants c0 and c1, with c1 positive, for which we can actually prove that

f1 (x) ≤ c0 + c1 * f2 (x)

for every x in the domain of those functions (which is to say for every non-negative real number x).
