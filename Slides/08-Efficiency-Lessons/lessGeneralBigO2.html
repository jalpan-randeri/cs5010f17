Theorem. If f: N → R and g: N → R are non-negative for all arguments n, and g is in O(f) according to the less general definition, then g is in O(f) according to the more general definition.

Theorem. If f: N → R and g: N → R are non-negative for all arguments n, and the constant function that always returns 1 is in O(f) according to the less general definition, and g is in O(f) according to the more general definition, then g is also in O(f) according to the less general definition.

In practice, computer scientists who use the less general definition write O(f) only when f is positive-valued and monotonic, which implies the constant-valued function that always returns 1 is in O(f).

Exercise: Can you give an example of a positive-valued function f: N → R for which (λ n . 1) ∉ O(f) according to the less general definition?

The two theorems above therefore prove that, so far as computer scientists are concerned, the less general definition is basically the definition you'd get by restricting the more general definition to functions from the natural numbers to the positive real numbers.
