Proof by Induction

We are going to use induction to prove the correctness of quicksort.

A proof by induction is organized around an induction hypothesis, which is sort of like an invariant that's parameterized by a non-negative integer n.

When proving the correctness of a program, the induction hypothesis typically says the program returns the correct result for all inputs whose size is no greater than n, and that's the kind of induction hypothesis we will use to prove the correctness of quicksort:

IH(n): If x is a list whose length is at most n, and less? is a total order defined on the elements of x, then (quicksort x less?) returns a sorted list of the elements of x.

Proofs by induction have the logical structure stated by the following rule of inference:

IH(0)
(∀ n) if IH(n) then IH(n+1)
————————————
(∀ n) IH(n)

That inference rule says: If you can prove the two statements above the line, then you are allowed to infer the statement below the line.
