A Less General Definition of O(f)

Many computer scientists use this less general definition of O(f):

Less General Definition. If f: N → R is a real-valued function of natural numbers, then

O(f) = { g: N → R | ∃ c, n0 > 0   ∀ n ≥ n0    g(n)   ≤   c ⋅ f(n) }
