Theorem. If f: N → R and g: N → R are non-negative for all arguments n, and g is in O(f) according to the less general definition, then g is in O(f) according to the more general definition.

Proof. Suppose g is in O(f) by the less general definition. Let c and n0 be positive constants such that for all n ≥ n0

g(n)    ≤    c ⋅ f(n)

Let c1 = c and let

c0 = max { g(n) | n < n0 }

Then for every n

  • if n < n0 then

    g(n)   ≤   c0   ≤   c0 + c1 ⋅ f(n)

  • if n ≥ n0 then

    g(n)   ≤   c ⋅ f(n)   =   c1 ⋅ f(n)   ≤   c0 + c1 ⋅ f(n)

So g ∈ O(f) by the more general definition. ∎
