A few more comments about Java:

In Java, the names of most data types start with a capital letter, but Java treats a small set of primitive data types, such as int, differently. The capitalized data types are called reference types. The un-capitalized data types are the primitive types.

As you have probably noticed by now, a method's return type is written in front of the name of the method, and the type of each argument is written in front of the name of the argument. That's a syntactic peculiarity Java copied from C.

Java copied many other syntactic peculiarities from C and other languages. Having become popular, Java is now a language many other languages are copying.
