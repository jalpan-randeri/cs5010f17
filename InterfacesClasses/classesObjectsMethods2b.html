Classes and Objects

  • A class definition is like a define-struct.
  • It specifies the names of the fields of its objects.
  • It may also specify some methods. Each method has a name and a definition.

For example: 

    class C2 {
        int a;
        int b;
        int c;
    
        public int foo () { return a + b; }
        public int bar (int n) { return c * n; }
        public int baz (int n) { return this.foo() + n; }
        ...
    }

Here is another example of a class: 

    class C {
        public int foo () { return 17; }
        public int bar (int n) { return 6 * n; }
        public int baz (int n) { return this.foo() + n; }
        ...
    }

The methods of class C are related to those of class C2, but class C doesn't have any fields.
